Air Quality Skill Privacy Policy

Your privacy is valued, so I want to be explicit about what kind of personal information is involved when you interact with Air Quality.

Air Quality skill allows you to add 'places' where air quality observations and forecasts will be created for. A place consists of place name and address (see this guide). You can also decide at what accuracy the address is given. All places will be persisted together with Alexa user ID so that places can later be found by the skill by using the user ID as a key. Only Air Quality skill can access this data provided that the request contains a valid user ID. Persisted places are not used for any other purpose. No other data is persisted by the skill.
