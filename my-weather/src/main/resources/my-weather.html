My Weather

Overview

My Wether is an Alexa skill which is able to read weather observations and forecats for places in Finland. Observation and forecast data is based on the Finnish Meterological Institute (FMI) open data, which is licenced based on CC BY 4.0 by FMI.

Weather observations are read from FMI's weather station or observatory closest to the place given by the user. FMI has over 400 stations so the coverage is good all over Finland. The weather forecast is based on RCR HIRLAM numerical forecast.

User can add his/her different addresses, i.e. 'places', which will be persisted and used by the skill when the user asks observations or forecasts for the given place. 

Configuration

Places

User can add address for the following place names:

home, work, school, gym, cabin, downtown, place1, place2 and place3

Places are added to Alexa's to-do list by using the following syntax:

place:<place_name>:<street_address>,<city>

For example:

place:school:Nissintie 2,Espoo

User then asks Alexa to save places:

Alexa, ask My Weather to save places.

Notice that prior to using My Weather skill, the user must grant list read and write permissions to the skill in the ALexa application. After places were persisted then corresponding to-do list items will be automatically removed by the skill. Adding a new place with the same name will overwrite the old place. 

If the user has not saved any places, then My Weather skill will get observations and forecasts for the capital city Helsinki.

Example Utterances

  • Alexa, open my weather.
    • This launches the interactive mode. Exit with exit, stop, good bye etc.
    • ...get observations
    • ...get forecast
    • etc. similar to one-shot commands listed below, but without Alexa, ask My Weather to...
  • Alexa, ask My Weather to get observations [for <place_name>]
    • Speaks the latest weather observations for the given place. If place is not specified then observations will be for home, provided that it has been saved (if not then Helsinki will be used).
  • Alexa, ask My Weather to get forecast [for <place_name> [at [tomorrow] <morning|noon|afternoon|evening|night>]]
    • Speaks the latest weather forecast for the given place. If place is not specified then forecast will be for home, provided that it has been saved (if not then Helsinki will be used).
